Substitution cipher

A substitution cipher is a simple way of encrypting a message. Each letter is substituted for a different unique character, according to a key. If the recipient has the key then they can decrypt the message by doing the substitution in reverse. Example: If the alphabet [a,b,c,d,e] is substituted with [d,a,b,c,e] then the message "badec" would be encrypted as "adceb".
