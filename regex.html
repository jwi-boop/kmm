Find the cities

via GIPHY

Let's have a look at some of the cities in Bangladesh, Nepal, Malawi, Mali and Niger. We will count how many of the big cities in this countries that contains the lettersequence chi. The lettersequence can not be the first ones in the cityname.

An Examples

Tchintabaraden is a city in Niger that contains chi, but Abala is not. The city Chikwawa in Malawi contains chi, but since they are the first letters in the name we will not count it.

You can write a regex using the tool below, another tool, or just be creative.

Regex

Just in case you're wondering. The regex is evaluated with both global and multiline flags.
